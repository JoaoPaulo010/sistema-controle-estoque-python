# Sistema de Controle de Estoque em Python

Projeto desenvolvido em Python para simular um sistema de controle de estoque.

## Funcionalidades

* Adicionar produtos
* Atualizar estoque
* Remover produtos
* Visualizar estoque
* Controle de quantidade e preços

## Tecnologias utilizadas

* Python

## Objetivo do projeto

Praticar lógica de programação, estruturas de dados, validações e manipulação de dicionários em Python.
